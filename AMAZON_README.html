About this Sample

  • Based on the 2009-03-31 API version.
  • This sample shows how to send an authenticated REST/Query request to the Amazon Product Advertising API using Java

Prerequisites

  • Amazon Product Advertising API Account (http://associates.amazon.com/gp/advertising/api/detail/main.html/).
  • Java version 1.5 or later
  • Apache Ant version 1.7 or later
  • The following java libraries:
    1. Apache Commons Codec 1.3 http://commons.apache.org/codec/ (included with this sample)

Quick Start

Sample comes with a SignedRequestsHelper class with the signature logic and a ItemLookupSample class that uses the SignedRequestsHelper to perform a simple ItemLookup operation. To get started with the sample, follow these steps:
  1. Log in to the Amazon Web Services portal at http://aws.amazon.com/ and copy your AWS Access Key ID and your AWS Secret Key. You can find them under the Your Account -> Access Identifiers menu after you log in.
  2. Extract the zip file into a working directory.
  3. Open com/amazon/advertising/api/sample/ItemLookupSample.java in an editor and change the AWS_ACCESS_KEY_ID and AWS_SECRET_KEY values to match the ones you obtained from AWS.
  4. Run the sample: 
    ant run
